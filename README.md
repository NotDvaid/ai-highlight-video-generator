# AIvideoGenerator
This project proposal outlines the design and development of an AI-powered Highlight Video Generator, a tool designed to assist nonprofits and community organizations in creating short-form video content efficiently and cost-effectively. Develop an MVP (Minimum Viable Product) that enables users to upload video and photo content, provide minimal context, and receive a polished 30–60 second highlight video using AI-powered selection and pre-built templates.
